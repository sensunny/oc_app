### Change Resize Mode:
```json
"splash": {
  "resizeMode": "cover"  // or "contain" (current)
}
```

Options:
- `contain` - Image fits inside, maintains aspect ratio
- `cover` - Image covers entire screen
- `native` - Uses native scaling
